# SKJ check data points

* Check state
* Check progress (%)

* Registration
  * Registration number
  * Status (e.g. 'registered', 'unsubscribed')
  * Date
    * First registration date
    * Start of current registration period
    * End of current registration period (expiry)
    * Unsubscribe date
  * Registration periods
    * Start
    * End
  * Re-registration submitted (yes / no)
  * Expired and re-registration submitted (yes / no)

* Register
  * Name (e.g. 'Jeugd- en gezinsprofessional SKJ / HSS SW Jeugd')
  * Level
  * Shown in register (yes / no)
  * Phone
  * Address
    * Street
    * House number
    * House number addition
    * Postal code
    * City

* Name
  * Display name (abbreviated, e.g. 'W. de Bruijn' or 'C. (Winters) Schuurbiers')

* Job title

* Company city

* Disciplinary measures
  * Type (e.g. 'suspension')
  * Label (e.g. '(Voorwaardelijke) schorsing')
  * Date
    * Start
    * End
  * Reason

* Check (assertion)
  * Name initial matches first character of personal name from ID (yes / no)
  * Family name matches family name from ID (yes / no)
  * Registration status is 'registered' (yes / no)
  * Registration end date is in the future (yes / no)
  * No currently active disciplinary measure (yes / no)
  * Overall valid (yes / no)
  * Violation code (empty string if no violation)
